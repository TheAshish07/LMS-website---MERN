# 📚 LMS Website (MERN Stack)

A full-stack Learning Management System (LMS) built using the MERN stack, designed to provide a seamless learning experience for students and efficient management for admins.


## 💻 GitHub Repository
🔗 https://github.com/TheAshish07/LMS-website---MERN.git 

---

## 🧠 Features

- 🔐 User Authentication (Clerk)
- 👨‍🎓 Student Dashboard
- 📚 Course Listing & Management
- 🧑‍🏫 Role-based Access (Admin/Student)
- 📱 Responsive UI
- ⚡ Fast and optimized frontend with React

---

## 🛠️ Tech Stack

**Frontend:**
- React.js (JSX)
- Tailwind CSS

**Backend:**
- Node.js
- Express.js

**Database:**
-MongoDB for now / Will be changed to PostgreSQL later

**Authentication:**
- Clerk

**Deployment:**
- Vercel (Frontend)
